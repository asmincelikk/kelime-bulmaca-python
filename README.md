# Kelime Bulmaca Oyunu

Bu, Python ile yazılmış basit bir kelime tahmin oyunudur.  
Bilgisayar Türkiye'nin şehirlerinden gizli bir kelime seçer ve oyuncu, harf harf tahmin ederek bu kelimeyi bulmaya çalışır.

## Nasıl Çalıştırılır?

1. Bilgisayarında Python 3 yüklü olmalı.
2. Proje klasöründeki dosyaları indir.
3. Terminal veya komut satırını aç.
4. Oyunu başlatmak için "0" tuşuna bas.
